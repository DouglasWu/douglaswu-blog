改變人生的5個記憶秘訣

6月 22, 2017 分類 學習策略

「記憶力」是什麼？

一般人可能認為，記憶力就是一定時間內能塞多少東西進大腦，塞得越快、越多，就是記憶力越好。

那些能背到圓周率小數點後好幾百位的、撲克牌過目不忘的，大概都是大腦結構與眾不同的天才，我們只能望塵莫及。
真的是如此嗎？

因緣際會之下，我去參加了記憶大師陳光老師的課程，每週一天，持續四周的課程結束後，我得到一個結論：

每個人都可以成為「天才」，只要你用對方法。

我拿這套方法去做實驗，花了來回台北新竹的通勤時間，就把台北捷運的所有站名按照順序背起來；出了名難背的GRE單字，用兩個搭捷運上下班的時間就熟記50個；圓周率小數點後125位，正背、倒背、第幾位是什麼都有辦法回答。

在了解這套記憶學原理之前，從來都不知道自己也有這樣的能力。

而我做的只是換個方式記憶，把資訊換一種格式儲存進大腦。

現在我體會到，「記憶力」並不只是單純的塞東西進大腦。它是一種活用感官、聯想、邏輯推理的綜合能力，而當正確的把這些能力發揮出來時，你就會感覺到自己的記憶力瞬間飆升，學東西的速度變快，看事情的角度和格局也變得不同了。

上完陳光老師的課，萬般感動之餘，整理出五個實用的訣竅：用感官創造關聯、轉碼、建立索引定位、有效閱讀。以下分別介紹。

一、用感官創造關聯

請看以下15個名詞：
手機、咖啡、空拍機、房間、吸管、臉盆、大數據、美工刀、行動電源、雨傘、音響、塑膠袋、小狗、詠春拳、盆栽

如果要把這15個東西按照順序背起來，你要花多久呢？

一般人可能聽到要背，馬上就會開始重複默念：「手機、咖啡、空拍機…手機、咖啡、空拍機…」。但是用這個方法，會發現需要的時間非常久，因為你只是在記住每個字的音頻、節奏，訊號非常單一，因此也常常要搭配「默寫」，那花的時間就更長了。

稍微進階一點，可能會記每個字的開頭：「手咖空房吸 臉大美行雨 音塑小詠盆」。要記得字數變少了，速度會加快一點，但基本上和第一種一樣，也是利用音頻以及反覆念誦的方式對大腦加壓。而且這個方法有個風險：每個名詞只記第一個字，你怎麼知道「手」就是「手機」而不是「手套」呢？「大」為何一定要是「大數據」，不能是「大叔」嗎？另外，也許短時間內靠著音頻記起來了，一兩個小時之後再回想，還能準確的背出來嗎？很難，因為這些暫存的聲音記憶，對大腦而言並沒有任何意義。

如果你使用了正確的方法，30秒就能把這些名詞牢記，順背、倒背都沒問題，而且隔了好幾天都不會忘。

你要做的，就是跳脫「文字」和「聲音」的魔咒，釋放大腦的感官，讓這15樣東西活起來。

首先去感受每個物件的形象，例如「空拍機」，就要想像四個螺旋槳的遙控機，在空中飛翔的樣子，還會發出「滋滋」的聲音；至於「大數據」比較抽象的概念，可以把它具體化，想像有一堆數字在眼前飛來飛去；「詠春拳」當然就是葉問比出帥氣的姿勢，大喊：「我要一個打十個！」
再來，兩個兩個分別看，讓物件彼此產生關聯。

手機、咖啡：把手機泡到咖啡裡面
咖啡、空拍機：咖啡灑出來濺到空拍機
空拍機、房間：空拍機在房間裡飛
房間、吸管：房間裡有一根吸管
吸管、臉盆：用吸管去吸臉盆裡的水
臉盆、大數據：臉盆裡有一堆數字
大數據、美工刀：把一堆數字刻在美工刀上
美工刀、行動電源：用美工刀割斷行動電源的線
行動電源、雨傘：拿行動電源的線綁住雨傘
雨傘、音響：雨傘撐在音響上面
音響、塑膠袋：音響裝進塑膠袋
塑膠袋、小狗：塑膠袋套在小狗頭上
小狗、詠春拳：小狗去咬葉問
詠春拳、盆栽：葉問踢爆盆栽

讓物品兩兩產生關聯之後，神奇的事情就發生了，你根本不需要重複記誦，就能把15樣東西按照順序背出來，而且倒過來背也沒問題（不信你試試看）。

每個人的邏輯不同，為物品設定的關聯也不同。然而在設定關聯時，還是有些訣竅能讓你記得更清楚：

  1. 避免太長的故事：例如「大數據」和「美工刀」，如果設定成「小明在學大數據，一直學都學不會，感到很挫折，所以就用美工刀狂劃桌子洩憤」那就太長了！這樣還要花額外的精力去記這個故事。盡量設定成簡單、可以瞬間發生的事件即可。
  2. 要有方向性：行動電源、雨傘、音響，如果兩組都是「雨傘撐在行動電源上面」、「雨傘撐在音響上面」，那很容易方向錯亂，不知道誰先誰後。
  3. 越誇張越好：在設定關聯的時候，越鮮明、越生動的形象，能讓你產生更強的印象。舉例來說，「咖啡旁邊停著一台空拍機」跟「咖啡灑出來濺到空拍機」，哪個會讓你記的比較清楚呢？「吸管放到臉盆裡」和「用吸管去吸臉盆裡的水」哪個比較生動呢？

二、轉碼

（一）文字轉碼

上面舉的例子都是大家熟悉的名詞，可以很輕易的在腦海中產生形象。如果是聽都沒聽過、或者很難產生直接形象的專有名詞，例如人名、地名，那就要透過「轉碼」這個動作，利用諧音或者意象的聯想，賦予具體形象。

拿台北捷運板南線西邊數來十一個站名為例：頂埔、永寧、土城、海山、亞東醫院、府中、板橋、新埔、江子翠、龍山寺、西門。

「頂埔」可以轉換成「頂樓的花圃」；「永寧」則是「永遠不講話的人」；「土城」很直覺就是「土做的城堡」….依此類推，全部都轉換完成之後，就可以用第一個方法，把它們都串在一起：

頂樓的花圃躺著永遠不講話的人→永遠不講話的人被關在土做的城堡→海珊攻打土做的城堡→海珊受傷被送進中國(亞洲的東邊)的醫院→中國的醫院牆上掛著豆腐形狀的時鐘→用豆腐形狀的時鐘去砸老闆蓋的橋→老闆蓋的橋上堆滿新的樂譜→新的樂譜打開發現長江裡的翡翠→長江裡的翡翠拿去供奉山上有龍的寺廟→寺廟中的龍在瘋狂的吸門

是不是瞬間背起來了呢？

透過這樣的轉碼，不僅捷運站名，三十六計、六十四卦、各類領域專有名詞、新同學的名字、客戶的名字……都可以輕鬆的鎖在大腦。

（二）數字轉碼

所有可以記載的知識，不外乎三種型態：圖像、文字、數字。其中圖像最容易記憶，數字最難，因為單純的數字是沒有意義的，只是0~9十種符號的排列組合而已。

當你想要記一串數字（例如：148）的時候，如果只是不斷的重複默念：「148、148、148….」，那其實同時也在做一件事情：把184、418、481、814、841其他五種排列都忘掉。因此當你記超過七位數的時候會感到非常吃力，因為已經超過5000種排列組合。想牢記一組數字，絕對不能靠「無意識地重複」，而要用正確的方法，把數字「放」到大腦中。

而這個方法，就是「數字轉碼」。其實大家小時候都學過：
鉛筆1、鴨子2、耳朵3、帆船4、勾勾5、大肚6、拐杖7、眼鏡8、氣球9、棒球10

不要小看這簡單的數字形象轉換，搭配關聯設定，它可以讓你快速記住十個以內的物件。

例如2005年美國時代雜誌公布的十大延年益壽食物：

番茄、菠菜、花椰菜、蔓越莓(小紅莓)、大蒜、鮭魚、燕麥、堅果、紅酒、綠茶

將這十種食物，分別和十個數字形象設定簡潔的關聯：

鉛筆1：鉛筆插進蕃茄裡 →蕃茄
鴨子2：鴨子的腳會「撥」水 → 菠菜
耳朵3：耳朵上別了一朵小花 → 花椰菜
帆船4：小紅帽在划帆船 → 小紅莓
勾勾5：用勾子吊起一袋大蒜 → 大蒜
大肚6：鮭魚有大大的肚子 → 鮭魚
拐杖7：撐著拐杖在麥田中散步 → 燕麥
眼鏡8：戴著老花眼鏡的老爺爺在啃堅果 → 堅果
氣球9：酒紅色的氣球 → 紅酒
棒球10：棒球選手在採茶葉 → 綠茶

經過這樣的對應，第幾樣食物是什麼，都可以瞬間抓到。

然而，今天如果想要記一長串的數字，一位數的轉碼是不夠用的，必須採用二位數的轉碼，也就是00, 01, 02, …, 一直到97, 98, 99 都要有一個形象。

轉換有很多種方法，最常見的就是利用諧音，例如「08」轉換成「淋巴」、「57」轉成「武器」、「72」轉成「企鵝」；還可以看數字的長相，例如「00」看起來就像「眼鏡」兩個鏡片、「96」則像是飛舞的「蝴蝶」；另外有些數字本身就有文化上的意義，例如「51」可以轉成勞動節的「勞工」、「52」可以轉成一副有五十二張的「撲克牌」。

100個兩位數都轉碼完後，碰到一長串數字，例如：145926358993，首先把它切成兩兩一組，各自轉碼：

14（椅子） 59（五角大廈） 26（河流） 35（珊瑚） 89（斑鳩） 93（酒商）

再去創造這些字詞之間的關聯：

椅子放在五角大廈上 → 五角大廈下面有一條河流 → 河流裡面有珊瑚 → 珊瑚做的籠子裡面關著一隻斑鳩 → 斑鳩啄酒商

不過，如果要記憶的位數不是偶數，沒辦法兩兩分組，例如「159」、「26535」，該怎麼辦呢？我們需要更進階的三位數轉碼。

三位數就是二位數前面再加上一位數，我們把這0~9的一位數分別設定成以下的性質：

0 → 掛滿鈴鐺的
1 → 穿衣服的、包膜的
2 → 兩個
3 → 有麥當勞符號的
4 → 死掉的、壞掉的
5 → 跳舞的
6 → 光溜溜的、光亮亮的
7 → 漆上白色的
8 → 很多的（8諧音「發」，表示很多）
9 → 酒醉的、著火的

如此一來，「1 59」就是「包膜的五角大廈」，「26 535」則可以轉成「河流裡面有跳舞的珊瑚」。

學到這邊，再來看看圓周率到小數點後15位：

3.14159 26535 89793 （怎麼似曾相識？）
14 159 → 椅子放在包膜的五角大廈上
26 535 → 五角大廈底下有條河流，河流裡有跳舞的珊瑚
89 793 → 珊瑚做的龍子裡面有隻斑鳩，斑鳩飛去啄染白髮的酒商

沒有想像中的難記，對吧？

三、建立索引定位

你可能還記得「手機」到「盆栽」的15個名詞，但如果想知道第6個名詞是什麼？第10個呢？可能沒辦法很快地回答出來，只能從手機開始，或從盆栽倒推，一個一個數，中間某個連結斷掉，就無法繼續了。當你要記憶的東西數量遠大於15個時，這個問題會更加嚴重。

因此，需要一個方法，去定位每樣物件在整串序列中的位置。

大家已經知道如何運用1~10的數字形象記憶十種健康食物，依此類推，當然也能把1~15數轉碼後的形象，去記憶那15個名詞。但是那樣太慢了，不需要每個位置都設定，只要設定關鍵的幾個位置，其他相鄰的物件用關聯去推導就行。

首先，把15個物件分成五個一組，加上編號：

[1手機、2咖啡、3空拍機、4房間、5吸管]
[6臉盆、7大數據、8美工刀、9行動電源、10雨傘]
[11音響、12塑膠袋、13小狗、14詠春拳、15盆栽]

接著，挑出每組最中間的物件，將它和其對應的編號綁在一起：

3 → 山 → 空拍機飛到山上
8 → 爸爸 →爸爸在用美工刀
13 → 雨傘（「一三」的諧音） → 小狗在撐雨傘

設定好之後，當想知道編號10是什麼的時候，只要看三個已設定的編號中最靠近10的：8。我們已經知道編號8是美工刀，那編號10就是美工刀的下面兩個，利用關聯去回想：美工刀 (割)→ 行動電源 (綁)→ 雨傘，就知道編號10是雨傘囉！

為了更清晰的記住每個物件的位置，方便大腦查閱，陳光老師研發出「電震體」的結構，把最多25樣東西放進「五個圈圈」當中：

四、用已知導未知

許多人在學英文時，最痛苦的過程就是「背單字」。一個陌生的單字，會念會拼還不夠，可能還要在不同文章中看到過七次以上，才能把單字跟它的意思連結在一起，成為永久記憶。問題來了，如果今天的目標是在一週內熟記100個單字，那在七天內到底要看多少英文文章，才能讓這100個單字都成為「長期記憶」呢？

有考過GRE的朋友都說單字的題目最難，於是自認為英文程度還不錯的我，下載了GRE單字的app來體會到底難到什麼程度。打開app一看，超過90%的單字都沒看過，我利用零碎的時間背，背了七個就投降了。以往單字都是從文章、歌詞中學習，現在要把「背單字」這件事獨立出來，大腦瞬間當機。

那個app在我手機裡閒置了許久，直到學了陳光老師的記憶學，再度拿出來挑戰。效果之驚人連我自己也嚇到！利用上下班通勤的零碎時間，四天就熟記了150個！

這個秘訣就是：用已知導未知。

運用中文諧音記憶，是其中一個技巧。然而不是所有單字都能像sentimental（山東饅頭）、dilemma（地雷嗎）這樣有很好記的諧音。而且純用諧音有個風險，就是你縱使記得怎麼念，卻不一定記得怎麼拼。

這一套針對任何單字都通用的方法，有四個步驟：

  1. 確認單字發音
  2. 確認中文意思
  3. 將單字拆解成已知的片段
  4. 加入邏輯

直接拿GRE單字來舉例吧！

profligate (adj.) 放蕩的

首先要確定單字怎麼念。這一步很重要，因為發音準確，拼字才能準確。現在手機要查發音非常方便，下載個字典app就有了。

再來，確認中文的意思。不要只是記「放蕩的」這三個字，而要對這個詞有感覺，喚起它的形象、使用情境，可以藉助英文例句來確認。「放蕩的」就是揮霍無度，行為十分不檢點的意思。

下一個步驟，將單字拆成已知的片段。可以按照音節或是按照意思來拆。搜尋看看，profligate當中有沒有已經認識的單字？有，「gate」是「大門」的意思。剩下的按照音節拆，就變成「prof li gate」。

三個片段中，gate 有意義了，prof 和 li 還沒。這時要回想，你已知的英文單字裡，有哪些是「prof」和「li」開頭的？「professor（教授）」跟「lick（舔）」。

最後一個最神奇的步驟：加入邏輯。我們把 profligate 拆成了 「professor（教授）」、「 lick（舔）」、「 gate（大門）」。想像一下，堂堂一個教授，在用舌頭舔著大門，那模樣是不是非常「放蕩」，非常不檢點呢？

如此一來，就把一個生字記住了！

這絕對不是巧合，任何單字都可以藉由這四個步驟鎖在腦海中。我將會用另一篇文章詳細解釋這個方法，並且舉更多例子。

掌握了這個方法，背單字再也不需要重複默念、默寫，你只是用你「已知」的單字，靠邏輯的關係去認識「未知」的生字，不僅效率更高，而且更加輕鬆愉快。背單字搖身一變成為了發揮聯想力、創造力的遊戲。

五、有效閱讀

最後一個秘訣，在這資訊爆炸的時代尤其重要。你不一定要有過人的記憶力，卻需要培養資訊統整、系統思考的能力，將閱讀到的知識變成自己的，同時能夠辨別斷章取義的片面之詞。

有效閱讀有四個步驟：

  1. 刪除廢話
    大部分文章會為了易讀性、文字的通順，會加入一些連接詞、補充語句，或是把大家原本就懂的常識復述一遍。這些話讀過一次之後就要過濾掉，把更多心力、腦力花在整理文章的精髓知識上。
  2. 釐清大架構
    當你看到一個概念或一個事件時，要把格局拉得更大，從更深、更廣的角度看待這件事，一方面避免迷失於表象、人云亦云，一方面可以在腦中建立完整的知識架構。
  3. 5W分析
    任何資訊不外乎五個W：When、Where、Who、Why、What（天時、地利、人和、原因、結果）。在閱讀的時候要將零散的片段，歸納整理到這五個類別當中，如此一來能更有效地掌握一個事件的來龍去脈。
  4. 項目重排
    當文章作者在列舉項目的時候，順序可能是依照一些特定的邏輯，或是沒有任何邏輯。當你想把這些項目吸收變成自己的知識，你可以依照自己的邏輯，把它們重新排列成比較好記憶的順序。
    很多好的文章、書籍我們常常閱讀的當下覺得很充實，但隔了沒多久就淡忘了，這就是因為沒有經過「有效閱讀」，將資料轉換成可以永久吸收的資訊。然而，每篇文章、每本書的結構和風格都不太一樣，一時之間要改變原有的閱讀習慣也不是件容易的事，因此有效閱讀也是五個秘訣當中最需要練習的。

記憶是一種工具，它讓你能用更有效率的方式累積智慧。然而，想要人生有所改變、更接近自己理想中的狀態，還是要透過明確的目標設定與實際的行動。

陳光老師教的不只是記憶學，更多時候在倡導正面積極的人生態度。他本身是多間企業的經營者，時常鼓勵大家要用企業家的思維與胸襟去擁抱生命，為社會創造價值。

對我來說，這不只是記憶學的課程，還是一個自我成長的里程碑，非常推薦有興趣的人去深入了解！
